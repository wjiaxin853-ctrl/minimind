### 预训练命令
```bash
python train_pretrain.py \
  --dtype bfloat16 \
  --batch_size 32 \
  --accumulation_steps 8 \
  --max_seq_len 340 \
  --num_workers 4 \
  --log_interval 20 \
  --save_interval 500 \
  --use_wandb \
  --wandb_project MiniMind-Pretrain-2x3090 \
  --learning_rate 5e-4
```  

  | 参数 | 作者默认基线 | 你现在推荐值 | 说明 |
  |---|---:|---:|---|
  | GPU | 1 卡默认 | 2 张 3090 | 用 torchrun --nproc_per_node 2 |
  | device | cuda:0 | 由 torchrun 自动分配 | 多卡时不要手写单卡 --device |
  | epochs | 2 | 2 | 保持一致 |
  | batch_size | 32 | 8 | 两卡先保守，稳一点 |
  | accumulation_steps | 8 | 8 | 保持一致 |
  | 等效 batch | 32*8*1=256 | 8*8*2=128 | 你的推荐值更保守 |
  | learning_rate | 5e-4 | 5e-4 | 先不动 |
  | dtype | bfloat16 | bfloat16 | 3090 可用，优先这个 |
  | num_workers | 8 | 4 | 两卡够用，也更稳 |
  | max_seq_len | 340 | 340 | 保持一致 |
  | use_moe | 0 | 0 | 先不要开 |
  | data_path | ../dataset/pretrain_t2t_mini.jsonl | ../dataset/pretrain_t2t_mini.jsonl | 保持一致 |
  | log_interval | 100 | 20 | 更方便盯训练进度 |
  | save_interval | 1000 | 500 | 更频繁保存，适合远端训练 |
  | use_wandb | 默认关闭 | 开启 | 项目里实际走的是 swanlab 兼容接口 |
